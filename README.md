# Great Highway Academy — School Management System

A React + TypeScript + Tailwind app for running day-to-day school
operations: students, staff, attendance, fees, results, timetables,
inventory, and reporting.

## Features

- Student & staff records, attendance, and class timetables
- Fees, payments, bulk fee collection, expenses, and family statements
- Academic results and report cards
- Announcements, school calendar, and events
- Inventory and uniform tracking
- Branding and theme customization
- Reports centre and document templates

## Getting started

```bash
npm install
npm run dev
```

Demo login: `admin` / `admin123`

## Scripts

- `npm run dev` — start the Vite dev server
- `npm run build` — production build
- `npm run lint` — run ESLint
- `npm run typecheck` — run the TypeScript compiler with no emit
